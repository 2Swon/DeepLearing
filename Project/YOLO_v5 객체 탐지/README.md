# 자동차 번호판 객체탐지 후 모자이크

RealTime에 유리한 YOLOv5 모델을 사용하여 자동차 번호판 객체를 탐지하고<br>

yolo github의 detect.py 파일에 모자이크 코드를 추가하여 바운딩박스를 blur처리합니다.
